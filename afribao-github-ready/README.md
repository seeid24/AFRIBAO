# AfriBao — The African Capability Engine

> Rooted in Africa, built for the world. We build the digital foundation African businesses need to grow.

## About

AfriBao is a pan-African digital agency offering web development, business systems automation, AI integration, and strategic consulting to empower African businesses.

## Pages

- **Home** (`index.html`) — Hero, services overview, call to action
- **About Us** (`afribao-about.html`) — Team story, mission, vision
- **Our Solutions** (`afribao-solutions.html`) — Full service offerings
- **Pricing** (`afribao-pricing.html`) — Website packages and pricing tiers
- **Business Systems** (`afribao-systems.html`) — Automation and systems integration
- **Start Here** (`afribao-start.html`) — Getting started guide / onboarding
- **Contact** (`afribao-contact.html`) — Contact form and information
- **Our Work** (`afribao-work.html`) — Portfolio / case studies
